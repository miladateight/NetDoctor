# Net Doctor

Net Doctor یک برنامه گرافیکی ساده برای ویندوز است که مشکلات رایج اینترنت و شبکه را به زبان قابل فهم بررسی می‌کند.

برنامه ابتدا از کاربر می‌پرسد چه مشکلی دارد، سپس وضعیت اتصال، DNS، کیفیت شبکه، پورت‌ها، VPN و Proxy را بررسی می‌کند و نتیجه را ساده توضیح می‌دهد.

## قابلیت‌ها

- رابط کاربری انگلیسی و کاربرپسند
- تشخیص اتصال اینترنت داخلی کشور و اینترنت بین‌المللی
- بررسی DNS، latency، packet loss و TCP port
- تشخیص وضعیت VPN و Proxy
- دکمه Fix Safely برای اصلاح امن تنظیمات قابل برگشت
- پشتیبانی از Undo برای تغییرات امن
- اسکریپت ساخت فایل نصب با Inno Setup

## ساخت برنامه

```powershell
.\scripts\publish-win-x64.ps1 -SelfContained
```

## ساخت فایل نصب

```powershell
.\scripts\package-installer.ps1
```

خروجی نصب‌کننده در مسیر `artifacts\installer` ساخته می‌شود.
