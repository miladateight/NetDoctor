# Net Doctor

Net Doctor is a licensed Windows utility by Milad AT8 for checking and repairing common local network and internet connectivity problems.

This public release repository is intentionally release-only. It does not include source code, tests, build scripts, private licensing keys, or the local LicenseTool.

## Features

- Guided network diagnostics for Windows users.
- Checks for DNS, gateway, adapter, and connectivity problems.
- Repair-oriented workflow for common issues.
- Licensed commercial distribution.
